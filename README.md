# S2-3  A23-Express 基礎驗收

## 🔗開始使用

1. Clone這支Repository
```
git clone https://github.com/mirageapache/A23_Express.git

```
2.安裝Express
```
npm install express
```
3.啟動程式
```
npm run start
```
4.瀏覽網頁
```
開啟瀏覽器並輸入http://localhost:3000，即可瀏覽網頁
```
5.終止程式
```
在終端機按下兩次 ctrl + C 即可終止程式
```

## 💻開發環境
+ Express @4.18.2
+ Express-handlebars @6.0.6
+ nodemon @2.0.20
